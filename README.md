# Serverless API System (Pure)

A lightweight Serverless API system built with Vercel Functions and pure HTML/JS.

## Structure

- `/api/`: Serverless functions (Visitor, Hitokoto, Image)
- `/public/`: Static frontend (HTML/CSS/JS)
- `/config/`: Configuration files (Domain whitelist, content)

## Deployment

Deploy to Vercel directly. The `vercel.json` handles routing and file inclusion.

## Configuration

Edit files in `/config` to manage content.
